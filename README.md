/settax 20 - Setzt die Steuerrate auf 20% - Standard ist 10%
/settax getrate - Zeigt die aktuelle Steuerrate.
/settax topay - Zeigt zu zahlende Steuern an.
/settax reset - Setzt zu zahlende Steuern zurück.

Die Steuern werden bei jedem Besuch der Gildenbank eingezahlt und zurückgesetzt.
